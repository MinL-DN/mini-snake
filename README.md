## 源码目录介绍
```

├── component                    // 各实体
│   ├── camera.js                // 摄像机类
│   └── snake.js                 // 蛇类
├── public                       // 静态资源存放
├── scene                        // 场景
│   ├── index.js                 // 场景基础方法
│   └── main.js                  // 主场景
├── utils                        // 工具类
│   ├── libs                     // 系统工具
│   │   ├── runtime.js           // ES6 Symbol简易兼容（未使用）
│   │   ├── symbol.js            // ES6 Symbol简易兼容
│   │   └── weapp-adapter.js     // 小游戏适配器（未使用）
│   ├── dom.js                   // 定义游戏开发基础类
│   ├── gradientColor.js         // 渐变色工具
│   ├── index.js                 // 工具类入口
│   └── loadRe.js                // 获取静态资源
└── game.js                      // 启动文件

```
